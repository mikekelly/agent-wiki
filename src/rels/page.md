# wiki:page

Retrieves the current version of a single wiki page.

## Method

GET

## Response

Returns a HAL resource with the full page content including title, body,
tags, and version information.

## Links Available

- `wiki:edit-page` — Edit this page (creates a new version)
- `wiki:delete-page` — Delete this page
- `wiki:history` — View version history
- `wiki:pages` — Back to all pages
